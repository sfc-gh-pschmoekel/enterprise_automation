# Lab 3: Marketing Assistant

**Time:** 15 min | **Pattern:** Multi-tool (Analyst + Search + cross-department)

## Overview

A single agent with three capabilities: campaign analytics (Cortex Analyst), brand document search (Cortex Search), and inventory checks (calls Ops agent).

## Data Location

`HOL_AUTOMATION.MARKETING` — Tables: CAMPAIGNS, CAMPAIGN_PERFORMANCE, CHANNELS, AB_TESTS, BRAND_DOCUMENTS

## Your Task

1. Open **AI & ML → Studio → MARKETING_ASSISTANT**
2. Test with prompts that use different tools

## Sample Prompts

| Prompt | Tools Used |
|--------|------------|
| "How are campaigns performing?" | Analyst |
| "What are our promotional guidelines?" | Search |
| "Why is TikTok underperforming?" | Analyst |
| "Do we have inventory for Industrial Bearings?" | Ops Agent |
| "Can we run a flash sale on bearings? Check inventory and promo rules." | All three |

## Key Insight

One question can coordinate Marketing and Operations without a single email or meeting. The agent routes to the right specialist automatically.
